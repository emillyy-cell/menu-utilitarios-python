# # Menu de Utilitários em Python

Programa em Python com um menu de três funcionalidades, feito como exercício de lógica de programação durante o curso de Análise e Desenvolvimento de Sistemas (ADS).

## Funcionalidades

1. **Verificador de número** — informa se um número é par ou ímpar, e se é positivo, negativo ou zero.
2. **Calculadora** — realiza as quatro operações básicas (soma, subtração, multiplicação e divisão), com tratamento de erro para divisão por zero.
3. **Tabuada** — exibe a tabuada de um número escolhido, de 1 a 10.

## Como executar

Abra o arquivo `menu_utilitarios.py` em um editor Python (ou no Google Colab) e execute.

## Conceitos aplicados

- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`for`)
- Entrada e conversão de dados (`input`, `int`, `float`)
- Tratamento de erros simples (divisão por zero, opção inválida)
- Organização do código em funções
